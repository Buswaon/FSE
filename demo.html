BASIC TERMINAL TROUBLESHOOTING

No Display?

  • Ensure the monitor is powered on and plugged into a working electrical outlet.
  • Try using a different display cable or port.
  • Ensure the monitor brightness is not turned down or off.
  • Test the monitor with another device, like a different computer or laptop, to confirm that the monitor is working.
  • Inspect the VGA or HDMI cable for any looseness or defects. If the issue continues, try replacing the cable with a new one.
  • Verify that the monitor is functioning and that the power cable is properly connected. If the monitor still does not work, try to coordinate to tech team (FSE).

No Power?

  • Check the power supply and make sure that the power cable is plugged in securely also, the power switch is on and the power outlet is working.
  • Check for physical damage look for burn or a smell of burnt to if noticing any of it replace with a new power supply.

Auto Restarting?

  • Check for hardware malfunction/overheating.
  • If the terminal has Auto Voltage Regulator (AVR) or Uninterruptable Power Supply (UPS), check if it’s working properly.

Mouse not working?

  • Attempt to connect the mouse to a different port. If it still doesn’t work, replace it with a new or spare mouse.
  • Restart the PC to refresh USB drivers.

No internet connection?

  • Make sure the modem/router power cable is connected.
  • Try different LAN port
  • Inspect the modem for a red light indicator. If present, this indicates an issue with the Internet Service Provider (ISP).
  • Check the LAN cable if it is plugged in securely to the PC.
  • Check RJ45 lock if it’s still intact.
  • Check LAN cable if have a rat bite.
  • Reboot the switch or modem to refresh the unit.

RTG cannot proceed and displays a casino shell popup?

  • Go to Date and Time settings and change it to proper time and date.
  • Update display/graphic driver.

Unified Game Lobby can’t open?

  • If the UGL isn't working, uninstall Mozilla Firefox and then reinstall it from the official website. After installation, right-click on Firefox and select "Run as Administrator." Once it's running, customize Firefox as needed and re-enter the Token for the Unified Game Lobby.

Some games won’t open?

  • Replace new files/folder of the games that has problem.
